Auth API

로그인

POST /api/v1/auth/login HTTP/1.1
Content-Type: application/json;charset=UTF-8
Accept: application/json
Content-Length: 97
Host: localhost:8080

{
  "providerType" : "kakao",
  "identifier" : "user_id",
  "token" : "token_from_oauth_server"
}

Path Type Constraints Description

String

token

String

카카오 access token 또는 애플 identify token

GET /api/v1/spots?lat=37.6676198&lon=127.0468177&radius=5000 HTTP/1.1
Host: localhost:8080

Query Parameters

Parameter Optional Default Constraints Description

lat

lat(33.0 ~ 39.0), lon(124.0 ~ 132.0)로 입력해주세요.
좌표에 빈 값이 들어갈 수 없습니다.

위도

lat(33.0 ~ 39.0), lon(124.0 ~ 132.0)로 입력해주세요.

경도

radius

      "lon" : 127.0468177,
